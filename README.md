# ⚡ Glitch Energy Drink

Siberpunk estetiğiyle tasarlanmış, modern ve performans odaklı bir enerji içeceği landing page projesi.

## 🚀 Canlı Demo
[👉 Canlı Siteyi Buradan İnceleyin][(https://reecepyuksel.github.io/glitch-web-app/)]

## 🎨 Tasarım Detayları
* **Glitch Efektleri:** CSS animasyonları ile oluşturulmuş, markanın kimliğini yansıtan dinamik metin efektleri.
* **Modern Arayüz:** Tailwind CSS kullanılarak oluşturulmuş, "Glassmorphism" (cam efekti) barındıran şık bölümler.
* **Responsive:** Mobil ve masaüstü cihazlarla tam uyumlu (Responsive) yapı.
* **Ürün Serileri:** Original Root, Neural Link ve Kernel Panic gibi farklı segmentlere özel tasarlanmış ürün kartları.

## 🛠️ Kullanılan Teknolojiler
* **HTML5:** Yapısal içerik.
* **Tailwind CSS:** Modern ve hızlı stil yönetimi.
* **FontAwesome:** İkon desteği.
* **JavaScript:** Yumuşak kaydırma (smooth scroll) ve scroll reveal efektleri.
